# stamps-api
Stamps API is a backend application built using Spring Boot to manage and track stamping-related concerns in a manufacturing environment.
The system supports change request management, concern assignment, cost tracking, reporting, and document handling.

The application is deployed on Google Cloud Platform using Cloud Run for scalable serverless execution, with Cloud SQL as the database.
It is securely connected via Serverless VPC and exposed through a load balancer for efficient traffic management.

A complete CI/CD pipeline is implemented to automate build, test, and deployment processes, ensuring continuous delivery and reliability.
